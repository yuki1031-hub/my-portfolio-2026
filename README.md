# 偉人タイプ診断 🏛️

## はじめに

本リポジトリは、LINE LIFF 上で動作する「偉人タイプ診断」アプリケーションです。
ユーザーの回答に合わせて歴史上の偉人を判定し、結果画像を生成・表示します。

## 制作背景（作成した経緯）

私はこれまで、CBCV 事業において LINE プラットフォーム上での UI 実装（YAML 記述による Flex Message 等の構築）に従事してきました。

業務の中で「見た目の定義（YAML）」だけでなく、「その裏側で実際にアプリケーションを動かすロジック（機能）」までを一貫して実装できるエンジニアになりたいという思いが強くなりました。
そこで、React を用いたフロントエンド開発から、Vercel へのデプロイ、そして LIFF との連携まで、「コードを書いて実際に機能させる」という一歩先の技術を習得するために本アプリを作成しました。

## コンセプト

- 歴史 × 診断: 「もし歴史上の偉人が現代にいたら？」という親しみやすいテーマで、ユーザーが楽しみながら触れるコンテンツを目指しました。
- 直感的な UI: LINE のトーク画面のような馴染みやすい操作感を意識しています。

## 技術スタック

- Frontend: React, JavaScript
- Infrastructure: Vercel
- Platform: LINE LIFF
- Tools: GitHub, CodeSandbox

## 機能概要

1.  独自の点数制ロジック:
    単純なランダム表示ではなく、各回答にポイントを設定し、合計点数に応じた条件分岐によって最適な診断結果を判定するロジックを JavaScript で実装しました。
2.  結果表示:
    判定された偉人タイプに基づき、画像とテキストを動的に出し分けます。
3.  再診断機能:
    React のステート（状態）をリセットし、ページ遷移せずに最初からやり直せる機能を実装しました。
