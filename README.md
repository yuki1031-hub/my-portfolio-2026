##  収録プロジェクト (Included Projects)

このWebサイト内で、以下のアプリケーションが実際に動作します。

### 1. 性格診断チャットボット (Personality Diagnosis Chatbot)
* **概要**: ユーザーの回答に応じて、性格タイプ（指揮官、仲介者など）を診断する対話型アプリケーションです。
* **技術**: React (State管理), 条件分岐ロジック
* **ファイル**: `src/Diagnosis.js`

### 2. 広告コピー生成チャットボット (Ad Copy Generator Chatbot)
* **概要**: ヒアリング内容を元に、商品の広告コピーを自動生成するシミュレーションチャットボットです。
* **技術**: React, 非同期処理のシミュレーション
* **ファイル**: `src/AdChatbot.js`
